# ROS2 TurtleSim Task

## Task Objective
The goal of this task is to:
- Install ROS2 (Humble)
- Run the TurtleSim package
- Understand the basic concepts of ROS2:
  - Nodes
  - Topics
- Launch the turtlesim node
- Control the turtle using the keyboard
